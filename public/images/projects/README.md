# Project Images

Place project image files in this folder.

When you choose a file in the admin dashboard, the app stores the path as:

`/images/projects/<filename>`

For example:

`/images/projects/my-project-cover.png`

Important:
- The dashboard cannot physically upload files into this folder at runtime on Netlify/Vercel.
- You must copy the selected files into this folder in your local project before committing/deploying.
